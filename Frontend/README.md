# React + Vite
lệnh cài frontend

cài npm
npm init 

cài react router dom
npm install react react-dom react-router-dom

cài đặt vite react
npm install --save-dev @vitejs/plugin-react vite


Lệnh cài backend
npm init 

Cài đặt express
npm install express --save

cài đặt mysql
npm install mysql2 sequelize 

cài đặt json web token
npm install --save jsonwebtoken
 
 Backend: khởi động bằng đường dẫn (Ổ đĩa chứa)\Web_English\Backend\src>
 Khởi động bằng lệnh node server.js
 
 FrontEnd:
 Vào đường đẫn: (Ổ đĩa chứa)\Web_English\Frontend>
 Khởi chạy bằng lệnh npm run dev
 Hình ảnh sau khi upload từ máy sẽ lưu vào \Web_English\Backend\uploads
 .env: lưu MONGODB_URI, PORT và JWT_SECRET
